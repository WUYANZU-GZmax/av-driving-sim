# 自动驾驶三维道路场景仿真 (Autonomous Driving 3D Sim)

基于 **Three.js** 的自动驾驶三维仿真场景，可作为仿真测试的可视化基座，便于后续扩展传感器仿真与场景测试。

## 特性

- **路网结构**：双向 4 车道主路 + 两条交叉路，构成 **两个完整十字路口**（`z = ±90`），含停止线、四向斑马线、路口车道线断开处理
- **道路基础设施**：车道线（虚实线/双黄线/边线）、方向箭头、限速/停车/禁行标志、红绿灯（红黄绿周期切换）、路灯、行道树、建筑
- **自车模型**：车身 / 车窗 / 车轮 / 车灯 / 传感器支架，带真实车灯照射
- **交通流**：主路 + 横向交叉车流，配 LiDAR 环与目标检测框预览
- **运动控制**：`W/S` 加减速、`A/D` 转向、`Space` 手刹、`R` 重置、`C` 切换视角；左下角 UI 按钮同样可控
- **相机**：`OrbitControls` 始终启用，目标跟随自车，**鼠标拖拽旋转 3D 视角、滚轮缩放**
- **渲染**：方向光阴影 + 环境光 + 半球光 + 点光源路灯，ACES 色调映射 + Bloom 辉光，夜间风格

## 运行

静态页面，直接双击 `index.html` 即可在浏览器打开（需联网加载 Three.js / Tailwind CDN）。
或部署到任意静态托管（GitHub Pages / Vercel / Nginx 等）。
